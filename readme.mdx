# Gedeon Prototype

This is the repository for the Gedeon prototype website.

## Live Demo

You can view a live demo of the website at [gedeon-prototype.netlify.app](https://gedeon-prototype.netlify.app).

## Usage

To run the website locally, clone this repository and open the `index.html` file in your web browser.
